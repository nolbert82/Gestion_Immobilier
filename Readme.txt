Ceci est une application codé en Java et SQL dans laquelle un propriétaire peut gérer son parc immobilier en ajoutant, retirant et modifiant des locataires, immeubles, et bails dans une base de donnée. Le logiciel est sous la forme de MVC, en utilisant le patron Dao pour interagir avec les données.

L'interface est dynamique : elle affiche les attributs de n'importe quelle table de la BD à travers une seule vue. Les boutons pour modifier supprimer ou ajouter fonctionnent de la même manière.

Il est possible de générer des rapports dans lesquels se trouvent des calculs de charges, d'impots, de loyers impayés et autres. Les rapports sont générés au format .docx dans le dossier "documents". Vous y trouverais aussi la notice d'utilisation.
